Some details about Vault Sys

Sys is a smart anti-theft solution for lockers, combining hardware and software to deliver a security system that alerts users in real-time if their locker is accessed. Our system is powered by Arduino and MUX switches for precision circuit management, and it integrates seamlessly with a mobile app for user notifications and reporting suspicious activity.





THE PROBLEM

Locker thefts are a common problem, yet traditional locks fail to provide robust protection or real-time monitoring. We wanted to build something smarter—something that goes beyond a simple lock, giving students and admins peace of mind and true control.
